##  Deskripsi Proyek

**Kannban Sederhana** adalah aplikasi _to-do list_ berbasis **Flutter** yang dibuat untuk membantu pengguna mengelola tugas dengan lebih terstruktur.  
Proyek ini dibuat tanpa koneksi database maupun package tambahan — semua data disimpan secara lokal di memori aplikasi.  

Tujuannya adalah memberikan contoh sederhana tentang:
- Pembuatan UI interaktif dengan Flutter
- Pengelolaan state secara lokal
- Implementasi dasar fitur CRUD (Create, Read, Update, Delete)

---

##  Fitur Utama

✅ Menambah tugas baru  
✅ Menghapus tugas  
✅ Mengubah status tugas (To Do → In Progress → Done)  
✅ Tampilan sederhana dan ringan  
✅ Tanpa koneksi internet dan database
